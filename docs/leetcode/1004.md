### 解题思路
维护了一个滑动窗口，够简洁，但还算容易理解

### 代码
```java
int count = K;
        int ans = 0;
        int i = 0, j = 0;
        while (j < A.length) {
            if (count == 0) {
                ans = Math.max(ans, (j - i));
                i++;
                count += 1 - A[i - 1];
            }
            if (A[j] == 0 && count > 0) {
                ++j;
                count--;
                if (j < A.length && A[j] == 1) {
                    while (j < A.length && A[j] == 1) {
                        ++j;
                    }
                }
            } else {
                while (j < A.length && A[j] == 1) {
                    ++j;
                }
            }
        }
        ans = Math.max(ans, (j - i));
        return ans;

```
